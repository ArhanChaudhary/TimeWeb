      (THIS IS A WIP) Introduction
      Getting started Welcome! TimeWeb's user guide documents all of its features, quirks, and methodology. Reading this isn't required, but doing so will clear up confusion you may have and greatly help with your user experience. In this user guide, important notes and FAQs are marked and labeled. FAQ labels are used for questions you may have, and note labels are used for advice and important edge cases. An example of an FAQ label: Why doesn't the internal date change after midnight? This behavior isn't a bug nor a glitch, but is actually intentional. Changing the date at midnight will result in showing the work for the next day despite the current work day not being finished. So, the internal date only changes to the next at the hardcoded value of 4 AM. The "Table of contents" section to the left shows the complete outline of this user guide, and the "Important labels" section just below that lists every note and FAQ in this guide. If you came here because you're confused about something specific, there's a good chance you'll find your answer in either of those sections.
      What is TimeWeb and why should I use it? TimeWeb is a free, online time manager that prioritizes, sorts, and lists each of your daily school or work assignments. Unlike an ordinary planner, each of your assignments' work is split up and graphed over the days until they are due. Then, they are automatically prioritized so you will always know which assignment to do, when to do it, and how long it will take. TimeWeb's main focus is on customizability and convenience. As we'll get to later on in this guide, TimeWeb supports many advanced parameters that can be configured for you and your work style.       Assignment form
      What is the assignment form? The assignment form is what you'll use to input your assignments into TimeWeb. You can also use this form to edit assignments. Default form input values in the settings are accordingly set every time you make a new assignment, and they can be manually overridden.
      Standard inputs These inputs in the assignment form are mandatory for every assignment. There are 7 standard inputs:
      Name of this Assignment
      This input is simply the name of the assignment. This is always displayed at the left of the assignment header, and this is only used for you to differentiate one assignment from another.
      Date Assigned
      This input is the date in which the assignment was assigned. Dates from the future are also perfectly valid. If you don't remember, enter today's date instead.
      Due Date
      This input is the date in which the assignment is due. This can't be before the assignment date. If your assignment doesn't have a due date, leave this input empty, and a rough due date will be calculated based on your other inputs. Note that leaving this empty should be a last resort, as the predicted due date may not always be suitable.
      Name of each Unit of Work
      This input is how the assignment is split and divided up. As an analogy, books are divided up into pages. If you have no idea how to split and divide up the assignment, enter the special keywords "Minute" or "Hour" in this input. By doing this, you effectively divide the assignment up into units of time, which is usually easier to estimate. When to use "Minute" vs "Hour" "Minute" and "Hour" both accomplish the same task, acting as a fallback if you don't know how to split and divide up an assignment. Typically, you'll want to use "Minute" for working in short, precise intervals of a singular minute, and you'll want to use "Hour" for working in longer, less precise intervals of 60 minutes.
      Total number of Units in this Assignment
      This input is the number of units of work in the assignment. The "Unit" part of this input is replaced in the assignment form with the name of each unit of work. This number will be split up over the days in which the assignment is due.
      Total number of Units already Completed
      This input is the number of units of work you've already completed in the assignment. The "Unit" part of this input is replaced in the assignment form with the name of each unit of work. Since this represents the first point on the blue line and the initial work input, if you edit this value, all of your other work inputs are vertically translated accordingly.
      Estimated number of Minutes to complete each Unit
      This input is the estimated number of minutes to complete each unit in the assignment. The "Unit" part of this input is replaced in the assignment form with the name of each unit of work. Note that this value is logically locked to 1 when the name of each unit of work is "Minute" and 60 when the name of each unit of work is "Hour."
